### HOW TO

+ Setup your site on `_data/metadata.yaml`
+ Update Widget site on `_data/widget.yaml`
+ Update Navbar site on `_data/navbar.yaml`
+ Update Footer site on `_data/footer.yaml`
+ Update Home page site on `content/index.md`
+ Update All Page and Static Page on `content/page/*.md`
+ Update Blog Post on `content/blog/*.md`
+ Update Podcast Post on `content/podcast/*.md` - use iframe embed for podcast video and audio
+ Update and create team profile on `content/team/*.md`
+ Config blog list on `content/page/blog.md`
+ Config podcast list on `content/page/podcast.md`
+ Config web active list on `content/page/web_active.md`
+ Config web archive list on `content/page/web_archive.md`
+ Config Organization page on `content/page/organization.md`

### Widget Features

Simply switch false or true to show widget on your page

```
show_toc: false
show_post_list: false
show_podcast: false
show_about: false
show_web_status: 
  show: false
  show_archive: false
```

Colors

```
# using bootstrap color
color: primary
color: secondary
color: success
color: danger
color: warning
color: light
color: dark
```
